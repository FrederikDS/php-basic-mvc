# PHP Basic MVC

## Intro
A first attempt to build an basic MVC or Model View Controller pattern to get a grasp of it. 
Still WIPping it :) (Work in Progress)

## Goals
* MVP
    * Build a Basic MVC in PHP (LAMP Stack) as skeleton for simple web apps
    * MVC handles POST/GET HTTP methods
    * Rewrite URL with Apache2
    * Model interacts with SQL database(MySQL)
    * Basic Web app with user login (HTML/CSS/Bootstrap/JS)
    * Exception handling
* Nice to have
    * Handle AJAX requests
